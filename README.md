# 🍽️ Zomato Data Analysis Using Python

## 📌 Project Overview

This project analyzes restaurant data from Zomato to uncover customer preferences, restaurant performance trends, and factors influencing ratings and popularity. The analysis was performed using Python and various data analysis libraries to derive meaningful business insights through visualizations and exploratory data analysis (EDA).

---

## 🎯 Objective

The main objectives of this project are:

- Analyze restaurant ratings and customer preferences.
- Understand the impact of online ordering and table booking.
- Identify trends in restaurant popularity.
- Visualize key business metrics.
- Generate actionable insights for restaurant owners and food delivery platforms.

---

## 📂 Dataset Information

The dataset contains information about restaurants including:

- Restaurant Name
- Rating
- Votes
- Approximate Cost
- Online Order Availability
- Table Booking Availability
- Restaurant Type
- Other relevant attributes

## 📊 Project Workflow

### 1. Data Collection
- Imported Zomato dataset.

### 2. Data Cleaning
- Checked missing values.
- Removed duplicates.
- Converted data types where required.

### 3. Exploratory Data Analysis (EDA)
- Rating distribution analysis.
- Online ordering analysis.
- Table booking analysis.
- Restaurant popularity analysis.

### 4. Data Visualization
- Count Plots
- Histograms
- Bar Charts
- Pie Charts
- Correlation Analysis

### 5. Insights Generation
- Identified customer behavior patterns.
- Analyzed restaurant performance metrics.
- Derived business recommendations.

---

## 📈 Key Insights

- Most restaurants receive ratings within a specific range indicating customer satisfaction trends.
- Online ordering significantly impacts restaurant popularity.
- Restaurants with table booking often have better customer ratings.
- Highly rated restaurants generally receive more customer votes.
- Certain restaurant categories dominate the market.
